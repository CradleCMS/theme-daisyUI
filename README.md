# daisyUI-theme
Tailwind component design from daisyUI made into a Cradle CMS theme. 

* Index page components
* daisyUI color presets with theme settings support
* Open graph tags
* Production and Development mode

## Index page components
* Over-the-fold hero section
* Link Grid
* Image Grid
* Content from a page
* Banner sections
* Blogpost cards
* Slideshow
